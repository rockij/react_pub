# React Pub Storybook

Next.js와 Storybook을 통합한 React 컴포넌트 개발 환경입니다.

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
# Next.js 개발 서버
npm run dev

# Storybook 실행
npm run storybook
```

## 📁 프로젝트 구조

```
react_pub/
├── .storybook/          # Storybook 설정
├── src/
│   ├── app/            # Next.js 앱
│   └── components/     # 재사용 가능한 컴포넌트
└── public/             # 정적 파일
```

## 📖 문서

자세한 사용 가이드는 [STORYBOOK_GUIDE.md](./STORYBOOK_GUIDE.md)를 참고하세요.

## 🎨 컴포넌트

- **Button**: 다양한 스타일과 크기의 버튼 컴포넌트
- **Card**: 이미지와 내용을 표시하는 카드 컴포넌트

더 많은 컴포넌트는 Storybook에서 확인할 수 있습니다.
